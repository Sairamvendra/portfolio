# Sairam Vendra - Portfolio Website

A modern, accessible portfolio website built with Next.js 14, featuring a bold **Neobrutalism** design style.

## ğŸ¨ Design Style

This portfolio embraces **Neobrutalism** - a contemporary design movement characterized by:
- Bold, high-contrast colors
- Thick, hard borders (no rounded corners)
- Hard shadows (not blurred)
- Geometric shapes and brutalist typography
- Raw, unapologetic aesthetics

- **Fully Accessible**: WCAG 2.1 AA compliant with ARIA labels, keyboard navigation, and screen reader support

- **Smooth Animations**: Powered by Framer Motion
- **Video Showcase**: Embedded YouTube videos with autoplay functionality
